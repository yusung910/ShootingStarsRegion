# ShootingStarsRegion
 IOCP + UnrealEngine 포트폴리오입니다.


1. 사용 도구
 - 클라이언트 : UnrealEngine 4.26.2 C++ 프로젝트
 - 서버 : IOCP
 - DB : Mysql
 - 프레젠테이션 : https://drive.google.com/file/d/1YAqXUrS5XO_zLoEOpb7fW_8YX8y6dCml/view?usp=sharing

2. 구현 기능
 2.1 서버 구현 기능
   - 캐릭터 생성, 삭제
   - 계정 로그인
   - 채팅
   - 몬스터 AI 기능
 
 2.2 클라이언트 구현 기능
   - 카메라 스플라인 기능
   - 캐릭터 및 몬스터 전투
   - 캐릭터 상태 위젯 화면 표시
